## Manual TestCases
* offloadComputeTC
* offloadMediaDevicesTC
* offloadTizenHAMTC (Tizen Human Activity Monitor)
* offloadTizenPPMTC (Tizen Privacy Privilege Manager)

## Prerequisite
### On Client Device
1) Install SignalingServer.wgt on client device.
2) Run SignalingServer and "Start SignalingServer" button to start the server on client device.
3) Connect the client device to wifi access point.
4) Check the client device's IP address, e.g. by the steps below.
``` sh
$ sdb root on
$ sdb shell
$ ifconfig
```
### On Worker Device
1) Install OffloadWorker.wgt on worker device to use its hardware resources.
2) Run OffloadWorker, enter the client's IP address found adn click "Reload" button.
3) With the steps above, the connection between the client's signaling server and worker is established so that below test widget files can be tested.
4) Each test files has its own input tag to allow testers enter the server address they prefer, instead of the signaling server running in the client from step 2).
